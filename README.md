# 少儿乐高编程课程体系发展时间轴

一个展示3-6岁少儿乐高编程课程体系的交互式网页应用，包含四个学习阶段的详细信息和图片轮播展示。

## 项目介绍

本项目是一个完整的少儿乐高编程课程体系展示页面，涵盖从3岁到6岁的四个学习阶段：

- **意识世界**（3岁，小小班）- 乐高管道游戏套装
- **发现世界**（4岁，幼儿园小班）- 乐高螺丝刀套装
- **发明世界**（5岁，幼儿园中班）- 实物编程套装+机械齿轮套装
- **创造世界**（6岁，幼儿园大班）- Wedo2.0科学机器人套装

## 功能特性

- 📚 **详细课程内容** - 每个阶段的学习内容、主题方向、能力培养
- 🖼️ **图片轮播展示** - 每个阶段都有交互式图片轮播
- 📱 **响应式设计** - 支持桌面端和移动端访问
- 🎨 **精美UI设计** - 渐变背景、卡片式布局、悬停动画效果
- 🌐 **静态网页** - 无需服务器，直接在浏览器中打开

## 使用方法

### 方法一：直接打开（推荐）

1. 下载或克隆本项目到本地
2. 使用浏览器直接打开 `index.html` 文件
3. 即可浏览完整的课程体系

### 方法二：使用本地服务器

如果需要在局域网内访问或避免跨域问题，可以使用本地服务器：

**使用Python：**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然后在浏览器中访问：`http://localhost:8000`

**使用Node.js：**
```bash
# 安装http-server
npm install -g http-server

# 启动服务器
http-server -p 8000
```

## 部署说明

### 部署到GitHub Pages

1. 将代码推送到GitHub仓库
2. 在仓库设置中开启GitHub Pages
3. 选择 `main` 分支作为源
4. 访问：`https://your-username.github.io/repo-name/`

### 部署到Netlify

1. 访问 [Netlify](https://www.netlify.com)
2. 导入GitHub仓库
3. 点击部署，无需额外配置

### 部署到Vercel

1. 访问 [Vercel](https://vercel.com)
2. 导入GitHub仓库
3. 点击部署，无需额外配置

## 自定义修改

### 修改课程内容

编辑 `index.html` 文件，找到对应阶段的 `.content-list` 部分进行修改。

### 修改图片

替换对应的图片文件，保持文件名不变即可。

### 修改样式

编辑 `index.html` 文件中的 `<style>` 部分，可以修改颜色、布局、动画等。
